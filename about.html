About Me

Saya dulunya tidak mengenal apa itu dunia coding semenjak saya masuk ke SMK saya mulai mengetahui apa itu coding dikarenakan mendapatkan projek berbasis Arduino yang membuat saya tau apa itu coding dan sejak saat itu saya mulai tertarik dengan dunia coding yang membuat saya sekarang beda di sini sebagai mahasiswa IDB Bali dengan jurusan Sistem & teknologi Informasi.
